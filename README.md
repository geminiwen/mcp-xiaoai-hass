# MCP XiaoAi HASS Server

Use Hass API to control XiaoAi

# Installation

Recommend use `uvx` to setup

## requirements
We need cliclick to emulate mouse click event
> brew install cliclick

Use uvx to install this packages
> uvx mcp-xiaoai-hass


# Configuration & Usage
